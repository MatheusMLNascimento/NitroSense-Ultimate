                    KEY ALGORITHMS
================================================================

✅ Thermal Derivative:     dT/dt calculation for anticipation
✅ Exponential Backoff:    2^n retry delay (0.5s base)
✅ Rolling Average:        5-point temperature smoothing
✅ Color Interpolation:    Temperature → color mapping
✅ Emergency Protocol:     95°C threshold with process kill
✅ Watchdog Detection:     RPM=0 at T>75°C detection

================================================================
                  DATA STRUCTURES
================================================================

Deques (Sequential Storage):
  ├── temp_history[30]     - Last 30 temperature readings
  ├── rpm_history[30]      - Last 30 RPM readings
  ├── temp_cache[5]        - Recent temps for averaging
  ├── gpu_cache[5]         - GPU temps
  └── cpu_cache[5]         - CPU usage

Dictionaries (Key-Value Maps):
  ├── THERMAL_CONFIG       - Thermal curve settings
  ├── PROCESS_PROFILES     - Gaming/office/video profiles
  ├── COLOR_SCHEME         - UI color constants
  ├── TEMP_COLORS          - Color mapping
  ├── ERROR_MESSAGES       - Error code descriptions
  ├── SYSTEM_PATHS         - /sys/ paths
  └── CONFIG (JSON)        - User preferences

Classes:
  ├── ConfigManager        - Configuration persistence
  ├── HardwareManager      - EC/NBFC access
  ├── HardwareWorker       - Background monitoring
  ├── MonitoringEngine     - Sensor data collection
  ├── PredictiveAIEngine   - Thermal algorithms
  ├── FanController        - Fan speed commands
  ├── NitroSenseApp        - Main window
  ├── HomePage/StatusPage  - UI pages
  └── ErrorDialog          - Exception display

================================================================
              EXECUTION FLOW
================================================================

1. main.py
   └─> check_prerequisites()
       └─> verify Python, device, dependencies
   └─> ConfigManager() [Singleton]
       └─> load ~/.config/nitrosense/config.json
   └─> HardwareManager()
       └─> _load_ec_module()
       └─> _check_nbfc_service()
   └─> NitroSenseApp(hardware, config)
       └─> create 4 pages
       └─> start update timers
       └─> show main window

2. Runtime Loop
   ├─> HardwareWorker thread (1500ms)
   │   └─> run_protected_command() [with semaphore]
   │   └─> emit update_signal
   ├─> UI update_timer (2000ms)
   │   └─> MonitoringEngine.get_system_metrics()
   │   └─> PredictiveAIEngine.calculate_required_speed()
   │   └─> FanController.set_fan_speed()
   │   └─> update labels & graphs
   └─> GC every 100 cycles

================================================================
                   DEPLOYMENT
================================================================

Installation:
  $ python3.12 -m venv venv
  $ source venv/bin/activate
  $ pip install -r requirements.txt

Running:
  $ sudo python3 main.py

Logs:
  $ ~/.config/nitrosense/logs/nitrosense.log

Configuration:
  $ ~/.config/nitrosense/config.json
