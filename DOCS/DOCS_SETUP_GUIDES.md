# NitroSense Ultimate — Setup & Guides

Este arquivo agrupa os documentos do projeto destinados à instalação, uso inicial e guia rápido.

## Documentos incluídos

### `README.md`
- Guia principal do usuário.
- Contém descrição de recursos, requisitos, instalação, arquitetura e solução de problemas.
- Focado em Ubuntu 24.04 e Acer Nitro 5.

### `QUICKSTART.md`
- Guia rápido de instalação e execução.
- Passo a passo para configurar ambiente virtual, instalar dependências e iniciar o app.
- Ideal para primeiros testes e deployment imediato.

### `STEP_BY_STEP_GUIDE.md`
- Guia de implementação para correções críticas.
- Descreve alterações de código em `main_window.py`, `hardware/manager.py`, `config_page.py` e `labs_page.py`.
- Útil para desenvolvedores que vão finalizar integrações pendentes.

### `00_START_HERE.txt`
- Ponto de partida do projeto.
- Normalmente contém instruções iniciais, visão geral e próximos passos rápidos.
- Use como referência para começar a analisar a base do projeto.
