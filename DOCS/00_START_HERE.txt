🎓 KEY ARCHITECTURAL DECISIONS

1. Error Code Pattern
   → Replaces exceptions with (ErrorCode, value) returns
   → Prevents crashes from propagating to UI
   → Allows fine-grained error differentiation

2. Semaphore Protection
   → All NBFC/EC access protected by QSemaphore(1)
   → Prevents race conditions on shared resources
   → Exponential backoff retry (0.5s × 2^n)

3. Thread Worker Pattern
   → Heavy operations on QThread, never block UI
   → Signal/slot communication with main thread
   → Automatic timeout recovery

4. Singleton Configuration
   → ConfigManager as single point of truth
   → Thread-safe with RLock
   → Atomic file writes (temp + rename)

5. Bounded Memory Buffers
   → 30-point deque for monitoring history
   → GC trigger at 500MB RSS
   → Result caching to reduce I/O

6. Professional Logging
   → RotatingFileHandler (max 5 files)
   → Colored console output
   → Persistent crash logs
