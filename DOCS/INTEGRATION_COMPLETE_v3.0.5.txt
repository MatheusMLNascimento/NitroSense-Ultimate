[TASK 2] Watchdog Integration in Monitoring Thread
──────────────────────────────────────────────────

File: nitrosense/core/monitoring.py

New Method: start_monitoring()
    def start_monitoring(self) -> Tuple[ErrorCode, str]:
        """Start monitoring thread with IdlePriority for <0.5% CPU budget."""
        self.monitoring_active = True
        self.monitoring_thread = QThread()
        self.monitoring_thread.started.connect(self._monitoring_loop)
        self.monitoring_thread.setPriority(QThread.Priority.IdlePriority)
        self.monitoring_thread.start()

New Method: _monitoring_loop()
    Main Loop:
      1. Get metrics (CPU temp, GPU temp, fan RPM)
      2. Update state_machine atomically (thread-safe)
      3. Update dirty_bit cache (for render optimization)
      4. **watchdog.heartbeat()** ← Signals "I'm alive"
      5. sleep(1.0) ← 1-second interval for IdlePriority

Watchdog Integration:
    # In MonitoringEngine.__init__:
    self.watchdog = watchdog  # Passed from system.py
    
    # In _monitoring_loop:
    if self.watchdog:
        self.watchdog.heartbeat()  # Called every 1 second
    
    # In system.py bootstrap():
    self.monitoring = MonitoringEngine(self.hardware_manager, self.watchdog)

Result:
    - If monitoring stops for >10 seconds → watchdog timeout detected
    - Emergency bus reset triggered automatically
    - System recovers without user intervention
